# Трекер привычек

Full-stack приложение на C#, ASP.NET Core, SQLite и HTML.

## Функции
- Добавление привычек
- Отметка выполнения
- Удаление

## Как запустить
1. Установите [.NET 8+](https://dotnet.microsoft.com/download)
2. Выполните в папке проекта:
   ```bash
   dotnet run
   ```
3. Откройте `index.html` в браузере
